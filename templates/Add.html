Ajout de nouvelles oeuvres

L'ajout de nouvelles oeuvres sur ce site est pour le moment bloqué par le manque de moyens alloués. Vous pouvez toutefois passer par mon mail pour qu'un ajout soit fait manuellement. A terme, l'objectif serait d'automatiser cette tâche afin que n'importe qui puisse intéragir avec la base de données contenant les oeuvres.
